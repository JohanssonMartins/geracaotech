Descrição da Atividade

Esta atividade tem como objetivo a criação de uma página web simples utilizando HTML semântico, aplicando de forma correta as principais tags da linguagem.

Objetivo

Criar uma página web simples e semântica, utilizando corretamente as tags HTML para estruturar o conteúdo de forma clara e organizada.

Instruções Detalhadas
1. Configuração Inicial

Criar um arquivo chamado index.html.

Estruturar o documento HTML básico utilizando a estrutura padrão: doctype, html, head e body.

2. Cabeçalho da Página

Dentro da tag head:

Adicionar a tag title para definir o título da página.

Adicionar a tag meta charset="UTF-8" para definir o conjunto de caracteres.

3. Estrutura do Corpo da Página

Construir a página utilizando tags semânticas adequadas.

Header

Utilizar a tag header contendo um título (h1) e um subtítulo (h2).

Navegação

Utilizar a tag nav contendo uma lista com itens e links.

Seção Principal (main)
Article

Adicionar um título (h2).

Adicionar um parágrafo (p).

Section

Adicionar um subtítulo (h3).

Adicionar uma lista não ordenada com itens.

Entrega

Salvar o arquivo index.html.

Abrir o arquivo no navegador para validar a estrutura.

Enviar o arquivo no campo de entrega ou colar o código diretamente na área disponibilizada.
